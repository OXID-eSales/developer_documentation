Project wide development, modules, themes and components
========================================================

Project wide development options, modules, themes and components are the containers
with which you can customize, extend or configure the OXID eShop.
They are the entry points for the use of the other
features that are described in the section :doc:`Tell Me Something About Topic X </development/tell_me_about/index>`.
